---

## 📊 What's Inside

| Analysis | Description |
|----------|-------------|
| 🏆 Team Wins | Most winning teams in IPL history |
| 🏏 Top Batsmen | Players with highest total runs |
| 🎯 Top Bowlers | Players with most wickets |
| 💥 Most Sixes | Big hitters of IPL |
| 🪙 Toss Effect | Does toss winning help? |
| 📅 Season Trends | Matches per season over the years |
